Real Estate Platform

A full-stack MERN Real Estate Platform where users can buy, sell, and manage properties.

Features:-
1.Authentication
2.User Registration
3.Login & Logout
4.Email Verification
5.Forgot Password & Reset Password

Property Management:-
1.Add Property
2.Edit Property
3.Delete Property
4.View Property Details
5.Property Approval System

User Roles:-
1.Wishlist
2.Property Inquiries
3.Contact Form
4.Real-time Chat
5.User Profile Management
6.Image Upload with Cloudinary

Tech Stack:-
Frontend
-React.js
-React Router
-Axios
-Context API
-CSS

Backend
-Node.js
-Express.js
-MongoDB
-Mongoose
-JWT Authentication
-Cloudinary
